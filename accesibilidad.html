Nuestro Compromiso

En Special Forces Electrician S.A.S. creemos firmemente que todos merecen acceso igualitario a nuestros servicios y contenido digital, independientemente de sus capacidades físicas o cognitivas.

Hemos implementado múltiples herramientas y características de accesibilidad para garantizar que nuestra plataforma sea usable por personas con diversas necesidades y preferencias.

Importante: La accesibilidad no es solo una característica, es un derecho fundamental.

Herramientas de Accesibilidad

Control de Texto

Ajusta el tamaño del texto para una mejor legibilidad según tus necesidades visuales.

Modos de Contraste

Cambia el contraste de colores para mejorar la visibilidad del contenido.

Ayudas de Navegación

Herramientas para mejorar la navegación y el enfoque en el contenido.

Controles de Audio

Funciones de audio y control de efectos visuales dinámicos.

Características Implementadas

Navegación por Teclado

Todos los elementos interactivos son accesibles mediante navegación por teclado usando Tab, Enter y teclas de flecha. Atajos disponibles: Alt+1 (contenido principal), Alt+2 (navegación).

Implementado
Compatibilidad con Lectores de Pantalla

Compatibilidad completa con NVDA, JAWS, VoiceOver y otros lectores de pantalla mediante etiquetas ARIA semánticas y estructura HTML apropiada.

Implementado
Alto Contraste y Modo Oscuro

Relación de contraste mínima de 4.5:1 para texto normal y 3:1 para texto grande según las pautas WCAG 2.1. Incluye modo oscuro para reducir la fatiga visual.

Implementado
Diseño Responsive

Adaptación completa a diferentes dispositivos y orientaciones, incluyendo soporte para zoom hasta 200% sin pérdida de funcionalidad.

Implementado
Contenido Semántico

Uso correcto de elementos HTML semánticos, estructura de encabezados jerárquica y etiquetas descriptivas para mejor comprensión.

Implementado
Control de Tiempo y Animaciones

Los usuarios pueden pausar, detener o ajustar límites de tiempo en contenido dinámico. Control total sobre animaciones y efectos visuales.

Atajos de Teclado

Navegación General
  • Alt + 1 - Ir al contenido principal
  • Alt + 2 - Ir a la navegación
  • Tab - Siguiente elemento
  • Shift + Tab - Elemento anterior
  • Escape - Cerrar menús/modales
Herramientas de Accesibilidad
  • Ctrl + + - Aumentar texto
  • Ctrl + - - Disminuir texto
  • Ctrl + 0 - Restablecer texto
  • Enter - Activar elemento
  • Espacio - Activar botón
